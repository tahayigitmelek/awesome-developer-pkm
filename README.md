# Awesome Developer PKM [![Awesome](https://awesome.re/badge.svg)](https://awesome.re)

A curated list of awesome tools, workflows, and materials for Developer Personal Knowledge Management (PKM) and building a developer's "Second Brain".

The goal of Developer PKM is to systematically record architectures, learning notes, code snippets, and project documentation so that they are easily accessible in the future. A true PKM system goes beyond individual tools. It is a method that connects information spread across chats, bookmarks, emails, wikis, and code repositories into a single, searchable, and interconnected knowledge graph. The resources below are organized not just by function but by how they help you capture, connect, and recall knowledge from every aspect of your digital life.

## Workflows & Methodologies

- [Building a Second Brain (BASB)](https://www.buildingasecondbrain.com/) - Tiago Forte's method for saving, organizing, distilling, and expressing ideas with digital tools.
- [CODE Method](https://fortelabs.com/blog/the-4-levels-of-personal-knowledge-management/) - A four-step framework: Capture, Organize, Distill, Express, for turning information into useful knowledge.
- [Johnny.Decimal](https://johnnydecimal.com/) - A system that organizes projects in a structured, numbered folder format. This helps avoid file clutter and makes navigation faster.  
- [Linking Your Thinking](https://www.linkingyourthinking.com/) - Nick Milo's method for linking notes through Maps of Content. This approach creates useful knowledge structures.  
- [Map of Content (MoC)](https://notes.linkingyourthinking.com/Cards/MOCs+Overview) - A method for creating "hub" notes that gather and connect "atomic" notes, improving navigation speed.  
- [PARA Method](https://fortelabs.com/blog/para/) - A universal information management system that focuses on Projects, Areas, Resources, and Archives.  
- [Zettelkasten Method](https://zettelkasten.de/) - A note-taking system that connects ideas using unique identifiers and atomic notes.
