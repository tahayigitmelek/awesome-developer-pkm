# Contributing

Contributions are highly valued! Here's how you can contribute:

1. Fork the repository.
2. Edit the `README.md` file to add your entry.
3. Submit a pull request.

## Guidelines

- **Relevance:** Entries should be directly related to Developer PKM (Personal Knowledge Management), "Second Brain" concepts, or markdown-based note-taking workflows for developers.
- **Search First:** Please check for previous suggestions before creating a new Pull Request.
- **Format:** Add the link in this exact format: `- [Project Name](http://example.com) - A short description that ends with a period.`
- **GitHub Links:** If the tool has a repository on GitHub, use the GitHub link rather than the website.
- **Ordering:** Order projects **alphabetically** within each category.
- **Formatting:** Ensure that the new entry doesn't contain trailing whitespace and try to submit only one link at a time.
